About URL Encoding

URL encoding (also known as percent-encoding) is a mechanism for encoding information in a Uniform Resource Identifier (URI) under certain circumstances.

When to Use:
  • Encoding special characters in URLs
  • Preparing data for HTTP requests
  • Encoding form data
  • Decoding received encoded data
Common Encoded Characters:
Space = %20
! = %21
# = %23
$ = %24
& = %26
' = %27
( = %28
) = %29
+ = %2B
/ = %2F
: = %3A
? = %3F
