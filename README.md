## üöÄ Installation Guide

### Local Development
1. Clone or download the repository
2. Navigate to the project directory
3. Start a local server:
   - Python: `python -m http.server 8000`
   - Node.js: `npx serve`
   - PHP: `php -S localhost:8000`
4. Visit `http://localhost:8000` in your browser

### Production Deployment
1. Upload all files to your web hosting provider
2. Configure your domain to point to the hosting server
3. Ensure `.htaccess` or server configuration allows proper routing
4. Test all pages and calculators for functionality
5. Verify SSL certificate installation

### Domain Connection
1. Purchase a domain name from a registrar
2. Update DNS records to point to your hosting provider
3. Configure domain in hosting control panel
4. Update canonical URLs in HTML files if necessary

## üí∞ AdSense Integration Guide

### Recommended Ad Placements

#### Header Ads
- Place leaderboard ads (728x90 or 970x250) in the header section
- Position below the navigation menu
- Ensure ads don't interfere with site navigation

#### Content Ads
- Place rectangle ads (300x250) between calculator sections
- Insert responsive ads within calculator containers
- Use in-content ads between FAQ sections

#### Footer Ads
- Place banner ads (728x90) in the footer section
- Consider multiple ad units with proper spacing
- Ensure ads don't dominate the footer

### AdSense Best Practices
- Follow AdSense policies and guidelines
- Maintain quality content standards
- Ensure proper ad-to-content ratio
- Test ad performance regularly

## üõ†Ô∏è Maintenance Instructions

### Adding New Calculators
1. Create calculator HTML file in `/calculators/` directory
2. Include proper meta tags and schema markup
3. Add calculator form with appropriate input fields
4. Implement JavaScript calculation functions
5. Add calculator to tools.html in appropriate category
6. Update sitemap.xml with new calculator URL
7. Add calculator to index.html if featured

### Adding New Blog Posts
1. Create blog post HTML file in `/blog/` directory
2. Include SEO-optimized title and meta description
3. Use proper heading hierarchy (H1, H2, H3)
4. Add FAQ section and conclusion
5. Include internal links to related content
6. Add structured data (Article schema)
7. Update sitemap.xml with new blog URL

### Updating Sitemap
1. Add new URLs to sitemap.xml
2. Set appropriate priority (0.5-1.0)
3. Set appropriate changefreq (daily, weekly, monthly, yearly)
4. Update lastmod date to current date
5. Validate XML syntax

### Updating Logo (Circular Style)
1. Create circular logo image (recommended 500x500px)
2. Save as PNG with transparent background
3. Place in `/images/` directory
4. Update logo references in HTML files:
   - index.html
   - tools.html
   - blog.html
   - contact.html
   - About page
   - All calculator pages
5. Update sitemap.xml image references
